## 简介


这里是Web3Club，成立于2023年2月7日。<br>
一个 独立自由的Web3开源社区 致力于 Web3相关中文知识的建设 和各种公共品实验 <br>
<br>



### 社群宗旨 永远关注知识和技术的进步，而不是价格



在此，我们希望为所有的对Web3未来感兴趣和欲为其“添砖加瓦”的朋友们一起,<br>
创造出更美好的Web3未来前景。
